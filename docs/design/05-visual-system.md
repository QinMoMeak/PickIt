# PickIt 视觉系统

## 1. 视觉总原则

- 基础界面保持干净、浅色、高对比
- 重点动作和重要价格使用高识别强调色
- 让用户感到“这是我的整理工具”，不是“这是一个促销商城”

## 2. 颜色系统

初始技能检索给出青绿主色与橙色行动色，这比高饱和玫红更适合工具型场景，因此采用如下主方案。

### 品牌与语义色

| Token | Hex | 用途 |
|---|---|---|
| `brandPrimary` | `#0D9488` | 主品牌、激活态、重点强调 |
| `brandPrimaryContainer` | `#CCFBF1` | 主色弱化背景 |
| `brandSecondary` | `#14B8A6` | 次级高亮、图表辅助 |
| `actionAccent` | `#F97316` | 主按钮、保存、导出、关键行动 |
| `surfaceBase` | `#F8FAFC` | 页面底色 |
| `surfaceCard` | `#FFFFFF` | 卡片底色 |
| `surfaceMuted` | `#ECFDF5` | 标签、辅助背景 |
| `textPrimary` | `#0F172A` | 主文本 |
| `textSecondary` | `#475569` | 次文本 |
| `borderSubtle` | `#DCE7EF` | 输入框、卡片边界 |
| `warning` | `#F59E0B` | 低置信提醒 |
| `error` | `#DC2626` | 错误与危险操作 |
| `success` | `#16A34A` | 成功反馈 |

### 状态色映射

| 状态 | 色值 | 用法 |
|---|---|---|
| 未购买 | `#0D9488` | 默认状态胶囊 |
| 正在使用 | `#2563EB` | 活跃中 |
| 使用完 | `#64748B` | 完成 |
| 已损坏 | `#DC2626` | 风险或损坏 |
| 已放弃 | `#7C3AED` | 结束或弃用 |

## 3. 排版系统

中文优先，不使用只支持拉丁字符的标题字体作为主字体。

### 字体建议

- 主字体：`Noto Sans SC` 或 Android 系统中文无衬线
- 英文数字辅助：`Rubik`

### 字号层级

| Token | Size | Weight | 用途 |
|---|---|---|---|
| `Display` | 32sp | 700 | 大数字、空态标题 |
| `TitleLarge` | 24sp | 700 | 页面主标题 |
| `TitleMedium` | 20sp | 700 | 模块标题 |
| `TitleSmall` | 18sp | 600 | 卡片标题 |
| `BodyLarge` | 16sp | 500 | 主体内容 |
| `BodyMedium` | 14sp | 500 | 常规说明 |
| `LabelLarge` | 14sp | 700 | 按钮、标签 |
| `LabelSmall` | 12sp | 600 | 辅助信息、时间、状态 |

### 数字排版规则

- 价格使用 `Rubik` 或等宽感较强的数字字体更利于扫读
- 金额整数和小数位可做轻微字重差异

## 4. 栅格与间距

### 页面边距

- 手机主边距：16dp
- 大模块间距：24dp
- 小模块间距：12dp

### 卡片规范

- 常规卡片内边距：16dp
- 大卡片内边距：20dp
- 列表卡片最小高度：92dp

## 5. 圆角与阴影

| Token | 数值 | 用途 |
|---|---|---|
| `radiusSmall` | 10dp | 输入框、筛选胶囊 |
| `radiusMedium` | 16dp | 常规卡片 |
| `radiusLarge` | 24dp | 顶部摘要卡、空态插画容器 |
| `radiusFull` | 999dp | 标签、状态胶囊、按钮 |

### 阴影原则

- 默认使用很浅的卡片阴影
- 主要通过底色层次和描边建立信息分层，不依赖重阴影

## 6. 图标系统

- 使用统一线性图标风格，建议 Lucide 或 Material Symbols Rounded
- 默认图标大小 20dp 或 24dp
- 关键动作用实心或填充态图标强调

## 7. 动效系统

### 动效原则

- 快，短，明确
- 只服务状态变化，不做炫技过场

### 建议时长

| 场景 | 时长 |
|---|---|
| 按压反馈 | 80ms |
| 筛选切换 | 160ms |
| 卡片进入 | 220ms |
| 底部表弹出 | 240ms |
| 成功提示 | 180ms |

### 动效细节

- 列表初次进入使用轻微上移 + 淡入
- 识别成功进入预览页使用内容分段加载
- 所有动效需支持 `prefers-reduced-motion` 等价降级策略

## 8. 组件气质

### 搜索框

- 宽圆角
- 低对比描边
- 聚焦后主色高亮

### 筛选胶囊

- 默认浅底深字
- 激活后使用主色底 + 白字

### 商品卡片

- 白底
- 清晰标题层级
- 价格与状态是视觉焦点

### 主按钮

- 胶囊型
- 采用 `actionAccent`
- 固定底部时需带轻阴影以区别内容层
